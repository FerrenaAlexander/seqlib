# seqlib
code for generating simulated sequences and performing basic genomic-oriented biostats

### installation

```
git clone https://github.com/apf2139/seqlib.git
cd seqlib
pip install . 
```

### usage in python API

```
import seqlib
seqlib.Seqlib()
```
